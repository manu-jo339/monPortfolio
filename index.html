⭐ PROJET PHARE

Déploiement & Sécurisation d'un Serveur Web Apache sous Ubuntu

Configuration complète d'un serveur web Apache sur environnement Linux Ubuntu en machine virtuelle. Le projet couvre l'installation, la configuration des hôtes virtuels, la mise en place des règles de sécurité, et les bonnes pratiques de durcissement (hardening) d'un serveur de production.

  • Installation et configuration d'Apache 2 sur Ubuntu Server
  • Mise en place de Virtual Hosts et gestion des permissions
  • Sécurisation : désactivation modules inutiles, headers HTTP sécurisés
  • Configuration du pare-feu UFW et des règles d'accès
  • Tests de vulnérabilité et validation de la configuration
